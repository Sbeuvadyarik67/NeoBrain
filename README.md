## 🚀 NeoBrain v1.2 — Потоковые ответы и улучшенный интерфейс

### ✨ Что нового:

- 💨 **Потоковые ответы** — текст печатается по буквам, как в ChatGPT (попробуй сам!)
- 🎯 **Переключатель "Поток"** — включай/выключай режим в один клик
- 🏷️ **Новые названия моделей** (теперь понятно с первого взгляда):
  - ⚡ Qwen 1.5b — **Молниеносная** (для быстрых ответов)
  - 💻 Qwen 7b — **Сбалансированная** (скорость + качество)
  - 🎭 Llama 3.1 8b — **Творческая** (для ролевых игр)
  - 🧠 Mistral 7b — **Умная** (для сложных задач)
- 🎨 **Улучшенный интерфейс** — поле ввода по центру, ровные отступы, приятный глазу дизайн
- ⚡ **Оптимизация кода** — убраны дубли, всё работает быстрее и стабильнее

### 🐛 Исправлено:
- Чат больше не съезжает влево
- Поле ввода теперь по центру
- Убраны дублирующиеся кнопки и элементы

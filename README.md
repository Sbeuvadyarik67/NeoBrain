# 🧠 NeoBrain

**NeoBrain** — это локальный AI-чат, бесплатная и приватная альтернатива Character.ai и ChatGPT.  
Ролевой AI-бот, симулятор личности и генератор диалогов на основе искусственного интеллекта.

Работает **полностью офлайн** через Ollama. Никаких серверов, никакой подписки, никакой передачи данных.  
Всё на твоём компьютере.

---

## 🔥 Ключевые возможности

### 🎭 Создание персонажей AI
Как в Character.ai, но без ограничений.  
Создавай уникальных ИИ-персонажей, настраивай их личности, характеры и стили общения.  
Идеально для ролевых игр, творчества и сценариев.

### 🧠 Ролевой AI-чат
Общайся с ИИ в реальном времени.  
Поддерживает потоковые ответы, диалоги и симуляцию личностей.  
Подходит для игр, обучения, генерации идей и развлечения.

### 🌡️ Кринжометр (температура ответов)
Регулируй креативность ИИ:  
- 1–3 — сухие, логичные, по делу  
- 4–6 — дружелюбные, с лёгким юмором  
- 7–8 — креативные, с сленгом и эмодзи  
- 9–10 — безумные, кринжовые, зумерские

### 🌍 Полная локализация
Интерфейс полностью переведён на 3 языка:  
- 🇷🇺 Русский  
- 🇬🇧 English  
- 🇺🇦 Українська

### 🎨 Стили интерфейса
Выбирай оформление под настроение:  
- Minimal — строгий минимализм  
- Vibes — вайбовый и мягкий  
- Cyber — киберпанк  
- Cloudy — облачный и светлый

### ⚡ Автозапуск Ollama
Не нужно запускать Ollama вручную.  
При старте NeoBrain всё делает автоматически.

### 📋 Копирование текста
Кнопка копирования появляется при наведении на любое сообщение.  
Удобно для цитирования и сохранения.

### 🔒 Приватность и безопасность
Никаких серверов.  
Никакой передачи данных.  
Всё работает локально на твоём ПК.

## 📦 Быстрая установка

```bash
# 1. Клонируй репозиторий
git clone https://github.com/Sbeuvadyarik67/NeoBrain.git
cd NeoBrain

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Установи Ollama (если ещё нет)
# Windows: https://ollama.com/download/OllamaSetup.exe
# macOS: brew install ollama
# Linux: curl -fsSL https://ollama.com/install.sh | sh

# 4. Скачай модель
ollama pull qwen2.5-coder:1.5b

# 5. Запусти приложение
python main.py
```
